## 1. 注册与登录

### 注册账号

1. 打开应用首页，点击顶部「注册」标签
2. 填写以下信息：
   - **昵称**：1-20 个字符，其他用户看到的名称
   - **手机号**：11 位数字，作为登录账号，不可重复
   - **密码**：至少 6 位
3. 点击「注册」按钮
4. 提示「注册成功，请等待管理员审核」

> 注册后账号状态为「待审核」，需要管理员批准后才能登录。

### 登录

1. 在登录标签页输入手机号和密码
2. 点击「登录」按钮
3. 如果是超级管理员首次登录，系统会弹出强制修改密码弹窗，修改后方可进入

## 3. 好友系统

### 添加好友

1. 点击好友区域右上角「+ 添加」按钮
2. 在弹窗中输入对方的昵称或手机号
3. 点击「搜索」
4. 在搜索结果中点击「添加」发送好友请求

### 处理好友请求

1. 点击好友区域中的「好友请求」链接
2. 在弹窗中查看待处理的请求列表
3. 点击「同意」或「拒绝」

### 删除好友

在好友列表中，点击对应好友右侧的「删除」按钮并确认。

## 4. 房间与游戏

### 创建房间

1. 在首页点击「创建房间」按钮
2. 系统自动生成 6 位房间号并进入房间
3. 将房间号分享给好友即可

### 加入房间

**方式一**：在在线房间列表中点击「加入」

**方式二**：在首页输入框中输入 6 位房间号，点击「加入」

### 游戏流程

```
加入房间 → 等待人数 ≥ 3 → 房主点击开始 → 轮流作画 → 结算排名
```

#### 开始游戏

- 房间人数达到 **3 人以上**后，房主点击「开始游戏」按钮
- 系统自动分配回合，每人轮流当画家

#### 作画（画家视角）

当轮到你作画时：

1. 画布上方会显示**你的词语**
2. 使用工具栏进行绘画：
   - **颜色选择**：点击颜色方块选择画笔颜色
   - **粗细调节**：细（2px）/ 中（5px）/ 粗（10px）
   - **清空**：清除整个画布
   - **撤销**：撤销上一步操作
3. 每回合 **30 秒**倒计时，倒计时结束自动进入下一回合

#### 猜词（观众视角）

当别人作画时：

1. 画布上方显示词语的**下划线提示**（如 `_ _` 表示两字词语）
2. 在右侧答题区输入框中输入你的答案
3. 按 Enter 或点击「发送」提交
4. 答对后系统自动提示，你和画家各得 **1 分**

#### 计分规则

| 行为 | 得分 |
|------|------|
| 猜对答案 | +1 分 |
| 画作被猜对（画家） | +1 分 |
| 猜错 | 0 分 |

#### 游戏结算

所有回合结束后，弹出排名面板：

- 按本局得分降序排列
- 显示本局得分和累计总分
- 点击「返回房间列表」回到首页

## 5. 排行榜

首页右侧展示全局排行榜：

- 显示所有用户的累计总分排名
- 前三名分别显示奖牌标志
- 分数自动累加，跨局保留

## 6. 管理员后台

管理员（ADMIN / SUPER_ADMIN）可在首页右上角看到「管理后台」入口。

### 用户管理

| 操作 | 说明 |
|------|------|
| 待审核 | 查看待审核用户列表，点击「通过」或「拒绝」 |
| 全部 | 查看所有用户 |
| 已拒绝 | 查看被拒绝的用户 |
| 注销 | 删除用户账号，不可恢复 |

### 词库管理

#### 添加词语

- **单个添加**：在输入框中输入词语，点击「添加」
- **批量添加**：在文本框中每行输入一个词语，点击「批量添加」

#### 删除词语

- **单个删除**：点击词语行的「删除」按钮
- **批量删除**：勾选要删除的词语，点击「批量删除」

> 默认词语（苹果、大象、太阳、飞机、钢琴）不可删除，其复选框为灰色禁用状态。

### 房间管理

| 操作 | 说明 |
|------|------|
| 暂停 | 暂停正在进行的游戏 |
| 恢复 | 恢复已暂停的房间 |
| 解散 | 强制解散房间，所有成员被移出 |

### 统计面板

顶部显示系统统计数据：总用户数、待审核数、已通过数、活跃房间、游戏中、词库数、在线用户。

## 7. 多人联机

本项目基于 Docker 部署，默认将 8080 端口映射到宿主机，其他设备可通过以下方式加入游戏。

### 同一局域网（推荐）

同一 WiFi / 路由器下的设备，直接访问宿主机的局域网 IP：

```
http://192.168.123.191:8080
```

> 可在宿主机终端执行 `ipconfig`（Windows）或 `ifconfig`（Mac/Linux）查看局域网 IP。

**防火墙设置**：如果其他设备无法访问，需在 Windows 防火墙中放行 8080 端口：

1. 打开「Windows 防火墙」→「高级设置」
2. 入站规则 → 新建规则 → 端口 → TCP → 8080 → 允许连接

### 不同网络（内网穿透）

如果朋友不在同一局域网（如手机用流量、在家远程），需使用内网穿透工具：

| 工具 | 命令 | 说明 |
|------|------|------|
| ngrok | `ngrok http 8080` | 生成公网地址，免费版每次地址变化 |
| cpolar | `cpolar http 8080` | 国内服务，速度快 |
| 花生壳 | 图形界面操作 | 适合不熟悉命令行的用户 |

### 云服务器部署

如需长期稳定运行，可将项目部署到云服务器（阿里云/腾讯云等），绑定域名后任何人可通过域名访问。详见 [部署指南](deployment.md) 中的「服务器生产部署」章节。

### 多人联机方式汇总

| 场景 | 方式 | 地址示例 |
|------|------|---------|
| 同一 WiFi | 直接访问局域网 IP | `http://192.168.123.191:8080` |
| 不同网络 | 内网穿透工具 | `https://xxxx.ngrok.io` |
| 正式上线 | 云服务器 + 域名 | `http://your-domain.com` |

## 8. 常见问题

### Q: 注册后无法登录？

注册后需要管理员审核通过。请等待管理员批准，或联系管理员处理。

### Q: 忘记密码怎么办？

目前需要联系管理员重置密码。

### Q: 房间最少几个人才能开始游戏？

至少 **3 人**，房主点击「开始游戏」后自动开局。

### Q: 游戏中掉线了怎么办？

重新登录后可以再次加入同一个房间（通过房间号），但当前回合的进度不会恢复。

### Q: 分数如何计算？

- 每局游戏中猜对 +1 分，画家被猜对 +1 分
- 分数累加到用户总分，永久保留，体现在排行榜中

### Q: 管理员可以玩游戏吗？

可以。管理员账号同时也是普通用户，可以创建房间、添加好友、参与游戏。

### Q: 默认管理员账号是什么？

| 字段 | 值 |
|------|----|
| 手机号 | `00000000000` |
| 初始密码 | `admin123`（首次登录强制修改） |

### Q: 如何查看 API 文档？

启动项目后访问 `http://localhost:8080/swagger-ui.html`
